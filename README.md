# React native template
> Template for react native applications for a quick start

[![NPM Version][npm-image]][npm-url]

Includeds:

* React navigation
* Gesture handles
* Eslint with prettier
* React hooks
* Reactotron
* Root import
* Styled components
* axios
* Prop-types

## Installation

OS X & Linux:

```sh
react-native init ProjectName --template raul-template
```
